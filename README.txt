Light Meter Application for Cameras
(Gives you ISO, Shutter speed and Aperture Settings)
Done by: Balamurugan Saiswaja

########################## USAGE ################################

1. Install Python 
2. Clone repo - git clone https://github.com/swajams/FilmCamera.git
3. Run python file 

########################## STEPS ################################

1. Main menu -                  Enter options 1 to start, 2 to view help menu or 3 to quit

2. Choose Lighting option (1) - Next choose the number that best represents
                                the lighting you want to shoot in.

3. ISO Setting -                Enter the ISO of your film you are shooting in

4. Additional Setting -         The program will display initial settings
                                you can choose to set a desired shutter speed (1)
                                or aperture (2)
